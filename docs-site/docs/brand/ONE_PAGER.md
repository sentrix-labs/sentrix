## Three differentiators

### 1. Bitcoin tokenomics + Ethereum dev experience

Sound-money supply discipline (fixed cap, halving) — the property that made Bitcoin durable — combined with the smart-contract programmability that made Ethereum the dominant developer platform. Same Solidity contracts you write for Ethereum work on Sentrix.

### 2. Indonesia-anchored, Southeast Asia scope

Built where blockchain solves real problems for the real economy — fractional asset ownership, transparent settlement, programmable financial primitives. Indonesia first (600M+ population in SEA) with regulatory pathway in active development.

### 3. RWA-native by design

Architected for tokenization of real-world assets. Voyager DPoS+BFT delivers institutional-grade settlement finality. EVM compatibility means existing tokenization standards (ERC-3643, ERC-1400) drop in without protocol changes.
